# 🤖 Trading Bot with Telegram & Google AI

## Features
- ✅ Telegram Bot Commands
- ✅ Google AI Market Analysis
- ✅ ATR Trailing Stop Loss
- ✅ Auto Capital Update
- ✅ 5-Minute Intraday Trading

## Commands
| Command | Description |
|---------|-------------|
| `/start` | Show menu |
| `/status` | System status |
| `/balance` | Account balance |
| `/pnl` | P&L summary |
| `/positions` | Open trades |
| `/market` | Market data |
| `/signals` | Trading signals |
| `/ai` | AI analysis |
| `/auto_on` | Auto trading ON |
| `/auto_off` | Auto trading OFF |
| `/mode` | Toggle Paper/Live |
| `/buy NIFTY 1` | Buy 1 lot |
| `/sell NIFTY 1` | Sell 1 lot |
| `/help` | All commands |

## Telegram Bot
- Token: Configured
- Admin ID: Configured

## Google AI
- Gemini Pro API integrated
- Market sentiment analysis
- Trading recommendations

## Deployment
1. Clone repository
2. Install: `pip install -r requirements.txt`
3. Run: `python trading_bot.py`
